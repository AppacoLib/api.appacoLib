cgi exemple :

dans cette exemple on créer une application CGI qui s'execute dans le server.
Le dictionaire est crier localement.
elle attend des requète http, après le traitement, les resultats sont envoyer où où client.

on utiliser directement le code source de la library.
le code source de la bibliothèque se trouve dans "../appaco_lib"

dans cette example on donne juste le nom (le chemin complet) de dictionnaire pour crier la structure,
ensuit on donne les mots requète.

on a un MakeFile qui compile le tout , l'exemple et le code de la bibliothèque en meme temps.
