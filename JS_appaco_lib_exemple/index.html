Appaco_Lib

APProximate AutoComplete Library

Appaco_Lib is an approximate autocompletion library which facilitates and speeds up input writing by offering a list of words (suggestions) which complements the few characters typed into the input text. It allows 1-edit error in the prefix entered by the user, and also to improve the quality of the results using a ranking system, according to the importance (score) of words. The library also permits static and dynamic ranking.

Appaco_lib features:

• exact autocomplete : list of suggestion contain only words that begin by the same prefix typed in text box.
• approximate autocomplete : we allow one error in the prefix typed in text box, which mean we return all word(s) that begin by the same prefix and also the word(s) that begin by a prefix similar to the prefix typed.
• static ranking : words will be displayed according to the importance of words.
• dynamic ranking : static ranking + update the score of the selected word (example +1)
• local dictionary source (javascript) or taken on a remote server, which allows querying a huge number of words in local (up to 200000 sentences, see as example English dictionary).
• very fast ajax queries to get the results from server (written in C/C++)


Supported language:

appaco_lib is coded in two languages : javascript and C/C++.
JS appaco_lib : used in client side, do all the work locally, and it can get result from server side.
C appaco_lib : can be used in any type of application, and can be use server side using the module FastCGI.
Those two libraries can be used separately.
